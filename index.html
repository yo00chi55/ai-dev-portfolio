How it works
制作フロー 🌸
1

Skill を設計する

SKILL.md にやり方・ルール・テンプレートを定義。Claude がいつでも参照できるナレッジベースを構築。

2

SubAgent に組み込む

フロントマターの skills: に Skill 名を指定し、専門エージェントとして定義。独立したコンテキストで動作。

3

Command で呼び出す

/write-article や /sns など、よく使うワークフローをスラッシュコマンド化。ワンコマンドで自動実行。

↑ この3ステップがClaude Code自動化の基本パターン ✦
